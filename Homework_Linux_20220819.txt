1. Что такое Linux и зачем она нужна.
 
   Linux — это операционная система для персональных компьютеров, которую можно использовать вместо Windows или Mac OS. Дистрибутивы Linux захватили рынок серверов, а рынок веб-серверов практически полностью. Использовать Linux на серверах выгодно, потому что эта операционная система бесплатна, вам не надо платить за лицензию и вы можете сразу же развернуть нужный образ Linux на сервере.

2. Чем отличается CLI от GUI.

CLI и GUI - это разные виды пользовательских интерфейсов. В основном они отличаются графикой, используемой в операционной системе. Для выполнения операции в системе CLI необходимо написать команду. С другой стороны, в GUI пользователи предоставили наглядные пособия (графика), которые включают в себя изображения и значки, что облегчает пользователям выполнение задачи напрямую.

3. Что такое SSH.

SSH (от англ. “Secure Shell”) — это протокол удаленного администрирования, разработанный для осуществления удаленного управления операционными системами.

4. Как сгенерировать пару RSA-ключей и зачем это делать.

- Открыть терминал и выполнить команду ssh-keygen -t rsa
- На консоль будет выведен вопрос. При ответе на этот и следующий вопрос просто нажать несколько раз клавишу Enter.
- После этого ключ будет создан в папке с:\users\usersname\.ssh\. Там будет создано два ключа: открытый и закрытый.

5. Как скопировать открытый ключ для пользователя USER на сервер SERVER. 

- Скопировать сгенерированный открытый ключ на сервер с помошью команды ssh-copy-id username@remote_host
- После этого увидим диалог, где нужно будет ввести пароль для учетной записи, к которой мы пытаетесь подключиться. Ввести пароль.

6. Какие задачи решаются с помощью команды 'gzip'.

Команда gzip предназначена для сжатия данных без потерь с помощью одноименной утилиты. Целью использования данной утилиты является экономия дискового пространства. Это достигается за счёт сжатия файла во время архивирования.

7. Как перенаправить вывод какой-либо команды в файл с целью сохранения.

Если файл не был создан: команда > имя файла (пример: echo "Hello world" > /tmp/text.txt)
Если файл с информацией уже создан и в него нужно дописать информацию: команда >> имя файла (пример: cat student.txt >> /tmp/text.txt)
